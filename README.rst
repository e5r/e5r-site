=========================
E5R Development Team Site
=========================

E5R Development Team web site.

Production: **http://e5rdev.com**
Staging: **http://www-stating.e5rdev.com**

Prerequisites
-------------

- Node >= 5.0
- NPM >= 3.0

Dev commands
------------

.. code-block:: text

   # Starting
   $ npm install
   $ npm run postinstall

   # Running
   $ build
   $ npm run start
